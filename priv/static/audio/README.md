# Audio Files for Chappal Chase

## Required Audio File

You need to add an audio file named `hole-hit.mp3` in this directory.

### File Details:
- **Filename**: `hole-hit.mp3`
- **Format**: MP3
- **Content**: Audio that plays when the center hole is hit
- **Suggested content**: Voice saying "Fuck me more!" or similar explicit audio

### How to Add:
1. Record or download your audio file
2. Convert it to MP3 format if needed
3. Name it `hole-hit.mp3`
4. Place it in this `/priv/static/audio/` directory

### Usage:
The audio will automatically play when players hit the center hole of the butt (not just the cheeks).

### Creating the Audio:
You can:
- Record your own voice
- Use a text-to-speech service
- Download royalty-free audio
- Commission voice acting

### Alternative Services for Text-to-Speech:
